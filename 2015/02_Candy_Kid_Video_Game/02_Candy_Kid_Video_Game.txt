<span style="font-family:verdana;font-size:85%;">
<a href="http://www.smspower.org/Scans/SegaComputer-Magazine-Issue02?gallerypage=17">Candy Kid</a> is a simple maze chase video game originally programmed by Grandstand Leisure from New Zealand in September 1984.  The game was written in <a href="http://en.wikipedia.org/wiki/BASIC">BASIC</a> programming language on Sega <a href="http://en.wikipedia.org/wiki/SG-1000#SC-3000">SC-3000</a>.  
<br /><br />
Inspired from previous posts on <a href="http://steveproxna.blogspot.co.uk/2013/06/sega-retro-gaming.html">Sega Retro Gaming</a> and <a href="http://steveproxna.blogspot.co.uk/2013/07/sega-basic-programming.html">Sega BASIC programming</a>, Candy Kid was re-written in <a href="http://en.wikipedia.org/wiki/XNA_(Microsoft)">XNA</a>.  However, the goal is to keep the super cool 80s retro arcade video game look and feel!

The game is now more a Pacman clone and also has the following features / options to vary game play:
<table width="100%">
<tr>
<td valign="top">
<ul>
<li>100 levels initially (if I can design) to 250??</li>
<li>Open or closed exits for different game play</li>
<li>Avoid trees that block or Death trees that kill</li>
<li>3x Candy Mamas "ghosts" more than original</li>
</ul>
</td>
<td valign="top">
<ul>
<li>God mode for invincibility to pass all levels</li>
<li>4x speeds: Slower, Medium, Faster, Insane</li>
<li>Option for 1x, 2x, or 3x ghosts chase player</li>
<li>Unlimited continues, increased lives choices</li>
</ul>
</td>
</tr>
