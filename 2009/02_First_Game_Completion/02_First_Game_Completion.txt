<span style="font-family:verdana;"><span style="font-size:85%;">Recently, I completed a basic 2D game to gain more experience in end-to-end game development on the XNA platform.The game, <a href="http://catalog.xna.com/en-US/GameDetails.aspx?catalogEntryId=2c11b9b8-5cb8-426e-b462-ecaa0827a151&type=2">Henway</a>, is basically a Frogger clone. The goal is to cross the road without being killed; hence this game is based on real life.<br/><br/><a href="http://catalog.xna.com/en-US/GameDetails.aspx?catalogEntryId=2c11b9b8-5cb8-426e-b462-ecaa0827a151&type=2">Henway</a> is built using content that I purchased in 2003. The game play is simple; which was important in order to concentrate on the end-to-end game 
development experience in XNA.
