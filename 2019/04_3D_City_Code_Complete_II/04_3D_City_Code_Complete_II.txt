<span style="font-family:verdana;font-size:85%;">
<a href="http://www.smspower.org/Scans/SegaComputer-Magazine-Issue16?gallerypage=6%2F7">3D City</a> is a simple "Shoot 'em up" video game originally programmed by StevePro Studios in New Zealand January 1988.  The game was written using <a href="http://en.wikipedia.org/wiki/BASIC">BASIC</a> programming language built against the Sega <a href="http://en.wikipedia.org/wiki/SG-1000#SC-3000">SC-3000</a>.
<br /><br />
In 2016, <a href="https://steveproxna.blogspot.com/2016/03/3dcity-code-complete.html">3D City</a> was re-written in <a href="http://en.wikipedia.org/wiki/C_(programming_language)">C</a> / <a href="https://en.wikipedia.org/wiki/Zilog_Z80">Z80</a> assembler to target the Sega Master System.  In 2018, 3D City was re-written in XNA to celebrate its 30yr anniversary and was ported to <a href="https://www.microsoft.com/en-us/p/3d-city/9nf638c5ll2z?activetab=pivot%3Aoverviewtab">Windows</a> [UWP], <a href="https://play.google.com/store/apps/details?id=com.steveproxna.x3dcity">Android</a> and <a href="https://apps.apple.com/us/app/retro-3d-city/id1441775916">iOS</a>.

<u>Summary</u><br />
3D City has come along way in 30yrs: from humble beginnings written in BASIC for the Sega SC-3000, to C / Z80 for the Sega Master System and finally re-written again for modern day desktop and mobile platforms.
