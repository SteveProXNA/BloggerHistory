<span style="font-family:verdana;font-size:85%;">
I am a big fan of <a href="https://en.wikipedia.org/wiki/The_Simpsons">The Simpsons</a>; particularly the early seasons of the show.  The Simpsons appeared in comic books, film, music and video games.  However, The Simpsons had not appeared on the <a href="http://en.wikipedia.org/wiki/Master_System">Sega Master System</a>!
<br />
Enter <a href="http://www.smspower.org/Homebrew/SimpsonsTrivia-SMS">Simpsons Trivia</a> Quiz.
<br /><br />
Inspired from previous posts on <a href="http://steveproxna.blogspot.com/2013/09/sega-console-programming.html">Sega Console Programming</a> and <a href="http://steveproxna.blogspot.com/2017/09/devkitsms-programming-setup.html">devkitSMS Programming Setup</a>, <b>Simpsons Trivia</b> is my third 8-bit video game written in <a href="http://en.wikipedia.org/wiki/C_(programming_language)">C</a> / <a href="https://en.wikipedia.org/wiki/Zilog_Z80">Z80</a> assembler to target the Sega Master System (SMS).

<u>Instructions</u>
<br />
Simple: move joystick Up and Down to select a multi-choice answer: A, B, C, D. 
<br />
Press Button1 to select an answer or progress forward through any prompts.
<br />
Note: Press Button2 to always go back. Joystick Left and Right are not used.
<br /><br />
<u>ROM Hacking</u>
<br />
You can hack this ROM!  Download + dump <a href="https://github.com/SteveProXNA/SimpsonsTriviaSMS/raw/master/SimpsonsTrivia-v1.02.sms">SimpsonsTrivia.sms</a> into Hex Editor, e.g. <a href="http://mh-nexus.de/downloads/HxDSetupEN.zip">HxD</a>, and modify bytes:
<table width="99%" border="1">
<tr>
<td valign="top" width="20%">&nbsp;ADDRESS</td>
<td valign="top" width="20%">&nbsp;VARIABLE</td>
<td valign="top" width="40%">&nbsp;DESCRIPTION</td>
</tr>
<tr>
<td valign="top" width="20%">&nbsp;<b>0x004F</b></td>
<td valign="top" width="20%">&nbsp;Debug</td>
<td valign="top" width="40%">&nbsp;Used in development => debugging.</td>
</tr>
<tr>
<td valign="top" width="20%">&nbsp;<b>0x0050</b></td>
<td valign="top" width="20%">&nbsp;Cheat</td>
<td valign="top" width="40%">&nbsp;Non-zero value will show answers.</td>
</tr>
<tr>
<td valign="top" width="20%">&nbsp;<b>0x0051</b></td>
<td valign="top" width="20%">&nbsp;Random</td>
<td valign="top" width="40%">&nbsp;Ask random questions during quiz.</td>
</tr>
<tr>
<td valign="top" width="20%">&nbsp;<b>0x0052</b></td>
<td valign="top" width="20%">&nbsp;Music</td>
<td valign="top" width="40%">&nbsp;0=music play otherwise off.</td>
</tr>
<tr>
<td valign="top" width="20%">&nbsp;<b>0x0053</b></td>
<td valign="top" width="20%">&nbsp;Sound</td>
<td valign="top" width="40%">&nbsp;0=sound on otherwise off.</td>
</tr>
<tr>
<td valign="top" width="20%">&nbsp;<b>0x0054</b></td>
<td valign="top" width="20%">&nbsp;Extra</td>
<td valign="top" width="40%">&nbsp;Set visibility on all extra info.</td>
</tr>
</table>
<pre class="csharp" name="code">
#ifndef _HACK_MANAGER_H_
#define _HACK_MANAGER_H_

#define PEEK(addr)   (*(unsigned char *)(addr))
#define POKE(addr, data) (*(unsigned char *)(addr) = (data))

#define HACKER_START  0x0050

extern unsigned char hacker_debug, hacker_cheat, hacker_random;
extern unsigned char hacker_music, hacker_sound, hacker_extra;

void engine_hack_manager_init()
{
  hacker_debug = PEEK( HACKER_START - 1 );  // 0x004F
  hacker_cheat = PEEK( HACKER_START + 0 );  // 0x0050
  hacker_random= PEEK( HACKER_START + 1 );  // 0x0051
  hacker_music = PEEK( HACKER_START + 2 );  // 0x0052
  hacker_sound = PEEK( HACKER_START + 3 );  // 0x0053
  hacker_extra = PEEK( HACKER_START + 4 );  // 0x0054
}
#endif//_HACK_MANAGER_H_
</pre>
<u>Cheat</u>
<br />
Some of the questions may be tough so you can cheat if necessary!  Either hack the ROM as above to show the answers or 
alternatively press Button2 five times on the Title screen when prompted to "Press Start".

<u>Retrospective</u>
<br />
In 2013, an initial ambitious <a href="http://steveproxna.blogspot.ie/2013/09/sega-console-programming.html">goal</a> was to build a retro game in pure Z80 assembler to target the Sega Master System.  During the past 5 years, it seemed more feasible to code in <a href="https://en.wikipedia.org/wiki/C_(programming_language)">C</a> using either the <a href="https://en.wikipedia.org/wiki/Z88DK">z88dk</a> or <a href="https://github.com/sverx/devkitSMS">devkitSMS</a>.
<br /><br />
However, the 2018 competition entry <a href="http://www.smspower.org/Homebrew/Astroswab-SMS">Astroswab</a> by <a href="http://www.smspower.org/forums/member6892">hang-on</a> <b>was</b> written in pure Z80 assembler.  By sharing the source code, this project has given some solid ideas on how it could be possible to achieve this goal <span style='font-family:Wingdings'>J</span><span><o:p></o:p></span>
